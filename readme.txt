== Installation ==

Installing this plugin as regular WordPress plugin.

After install, you are ready to use logo carousel in your post, page, widget or anywhere you like. How to create logo carousel? lets see.


1. Upload the folder team-freeÂ to "/wp-content/plugins/" '
2. Activate the plugin through the "Plugins" menu in WordPress .
4. Add member going to wp-admin -> Team -> Add New Member.
5. Go to Team > Shortcode Generator > Add New Shortcode> Publish.
6. Call shortcode to your WordPress editor directly.

Now save your page, then visit the newly created page & see something awesome. :)
